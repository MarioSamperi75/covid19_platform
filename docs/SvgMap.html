The class component SvgMap contains the Svg image (Sweden) and all its parts (regions). All the parts are interactive and receives a color as props.

An objects that contains data for the regions. Key: name of the region, Value: rgb (green) value.

A connection to the method getRegionNameFromMap(Layout). The method receives the name of the region and set the state selectedRegionObject.
